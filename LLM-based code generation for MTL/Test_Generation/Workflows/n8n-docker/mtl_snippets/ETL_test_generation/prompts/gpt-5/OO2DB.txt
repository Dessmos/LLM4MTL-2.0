    - `child` = child FK column in child table,
    - name = `<Child>Extends<Parent>`.

  - column name = attribute name,

  - creates FK column in referenced type’s table (`fkCol.table ::= r.type`),
  - FK col name `<referenceName>Id`, type `INT`,
  - FK direction: `parent = r.owner.equivalent().primaryKeys.first()`, `child = fkCol`,
