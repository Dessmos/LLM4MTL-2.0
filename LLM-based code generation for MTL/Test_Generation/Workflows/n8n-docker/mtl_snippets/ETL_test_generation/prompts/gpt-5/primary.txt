Rules in `primary.etl`:
- `Flowchart2Heading` transforms one `Flowchart!Flowchart` to one `HTML!DIV`.

  - Intent: DIV contains only headings of transitions, not links.
- `Transition2SourceLink` transforms each `Flowchart!Transition` to an `HTML!A`:
  - `a.value = t.source.name`
  - `a.ahref = "#" + t.source.name`
- `Transition2TargetLink` transforms each `Flowchart!Transition` to an `HTML!A`:
  - `a.value = t.name`
  - `a.ahref = "#" + t.target.name`
- `@primary Transition2Heading` transforms each `Flowchart!Transition` to an `HTML!H1`:
  - `h1.value = t.name`
