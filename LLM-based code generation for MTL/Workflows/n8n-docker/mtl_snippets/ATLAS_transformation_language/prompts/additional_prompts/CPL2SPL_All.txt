Create an ATL transformation module named **CPL2SPL** that converts models conforming to the **CPL** metamodel into models conforming to the **SPL** metamodel.  

Define helpers **statement**, **locations**, and **isSimple** to compute default statement nodes, location sequences, and to identify simple proxies.  

Include transformation rules **CPL2Program**, **SubAction2Function**, **Incoming2Method**, **Proxy2Return**, **Proxy2Select**, **Busy2SelectCase**, **NoAnswer2SelectCase**, **Redirection2SelectCase**, **Default2SelectDefault**, **SubCall2Return**, **Redirect2Return**, **AddressSwitch2SelectStat**, **SwitchedAddress2SelectCase**, and **Otherwise2SelectDefault** implementing the corresponding structural mappings.  

Also define helper rule **Location2URIConstant** to map a Location to an SPL URI constant.
